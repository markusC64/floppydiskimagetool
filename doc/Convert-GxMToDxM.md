# Convert-GxMToDxM

## SYNOPSIS
Converts a G1M,G2M or G4M disk image object to a D?M disk image object.

## SYNTAX

```
Convert-GxMToDxM [-gxmImag] <Object> [<CommonParameters>]
```

## DESCRIPTION
Converts a G1M,G2M or G4M disk image object to a D?M disk image object.

Cf Convert-G81ToD81 for Details
